# DiscoBunty v0.9.4

## New Features

- No new end-user features were added in this release.

## Improvements

- Hardened the WebUI form theme so shared input styling now explicitly controls text color, focus state, placeholders, select options, and dark-mode rendering.

## Bug Fixes

- Fixed white-background input rendering in the Core Configuration form so the Discord token, Guild ID, global roles, WebUI password, and power-control password fields remain readable.
- Fixed the add-server modal so the Port field and password-auth field also stay on the dark theme instead of falling back to a nearly unreadable light input style.
